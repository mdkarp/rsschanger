## Prerequisites

Install Flask:

```
$ pip install --user Flask
```

## Usage

```
$ python server.py
 * Running on http://127.0.0.1:5000/ (Press CTRL+C to quit)
```

The server is now available at [127.0.0.1:5000](http://127.0.0.1:5000/).
