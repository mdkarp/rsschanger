## Prerequisites

Install Flask:

```
$ pip install --user Flask
```

## Usage

Run `server.py` with Python:

```
# python server.py
 * Running on http://127.0.0.1:8080/ (Press CTRL+C to quit)
```

The server is now available at [127.0.0.1:8080](http://127.0.0.1:8080/).
